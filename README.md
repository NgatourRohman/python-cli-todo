# 📝 Python CLI To-Do List

Aplikasi To-Do List interaktif berbasis Command Line Interface (CLI) yang dibangun menggunakan Python dasar. Proyek ini dibuat untuk mempraktikkan logika pemrograman Python, operasi CRUD (Create, Read, Update, Delete), dan manipulasi penyimpanan data menggunakan format JSON.

## ✨ Fitur Utama

- **Dashboard Statistik:** Menampilkan persentase tugas selesai dan estimasi total sisa waktu pengerjaan.
- **Manajemen Tugas:** Tambah, edit, dan hapus tugas dengan mudah menggunakan pencarian ID unik.
- **Filter Tampilan:** Memfilter daftar tugas berdasarkan status (Selesai / Belum Selesai).
- **Penyimpanan Permanen:** Data otomatis disimpan dan dimuat dari file `tugas_saya.json`.
- **Penanganan Error (Error Handling):** Memastikan aplikasi tidak *crash* jika pengguna salah memasukkan tipe data pada terminal.
